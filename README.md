chessGame

Минимальнвя реализация кода который создает шахматную доску
реализует основные фигуры и описывает движение фигур
без графического интерфейса и искусственного интеллекта.

Основные классы
Spot – ячейка доски, содержит координаты и фигуру.

Piece – абстрактный класс для всех фигур.

Для каждой фигуры (Pawn, Rook, Knight, Bishop, Queen, King) создаются отдельные классы, наследующие Piece.

Board – основная доска 8x8 с методом инициализации начального положения фигур
